gpu_particles
==============
A simple python (with python-ogre) + C + OpenCL (GPU/CPU) implementation of simulation of gravitationally interacting particles. Supports Linux and Windows (with ports of gcc & make).

Requirements
--------------
- Windows/Linux
- AMD APP SDK
- gcc (or some similar)
- make
- python
- python-ogre

How to run
--------------
make && main.py
